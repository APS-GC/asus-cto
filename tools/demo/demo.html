Integration Code

To use this header in your third-party site, include:

HTML:

<!-- Include the web component script --> <script src="https://main--asus--aem-comp.aem.live/blocks/header/aem-header.js"></script> <!-- Use the component --> <aem-header fragment-url="https://main--asus--aem-comp.aem.live/nav.plain.html" base-url="https://main--asus--aem-comp.aem.live"> </aem-header>

JavaScript API:

// Get reference to the header component const header = document.querySelector('aem-header'); // Refresh the header header.refresh(); // Set user login state header.setUserState(true, 'John Doe'); // Update cart count header.setCartCount(3); // Listen for events header.addEventListener('aem-header-loaded', (e) => { console.log('Header loaded successfully'); }); header.addEventListener('aem-header-error', (e) => { console.error('Header error:', e.detail.error); });

CORS Configuration

For production use, ensure your EDS site has proper CORS headers configured:

# In your EDS project, create/update headers configuration: # .hlx/headers.yaml or similar headers: - key: Access-Control-Allow-Origin value: https://your-third-party-site.com - key: Access-Control-Allow-Methods value: GET, POST, OPTIONS - key: Access-Control-Allow-Headers value: Content-Type, Authorization
